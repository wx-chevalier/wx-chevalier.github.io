内存管理

所谓的内存管理（Memory Management），即是如何在计算机系统中协调与控制内存的使用，其往往会关注三个层次：MMUs, RAM 这样的硬件内存管理；Virtual Memory Protection 这样的操作系统级内存管理；以及内存的分配、垃圾回收等应用级的内存管理。
