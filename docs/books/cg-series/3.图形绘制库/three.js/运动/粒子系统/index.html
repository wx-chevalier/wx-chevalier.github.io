粒子系统

粒子

我们来看一下唯一一个我们一直忽略的概念:粒子( particle)。使用粒子可以很容易地创建出很多细小的物体,可以用来模拟雨滴和雪花。你也可以用粒子来创建有趣的三维效果。例如,你可以将某个单一几何体渲染成一组粒子,并分别对它们进行控制。
