Electron OpenSource List

Showcase

  • 2016-electron-python-example : Electron as GUI of Python Applications.

  • 2017-electron-process-reporter : Utility to extract interesting process reports of an Electron application.

  • 2017-electron-util : Useful utilities for Electron apps and modules.

  • 2018-PicGo : 🚀A simple & beautiful tool for pictures uploading built by vue-cli-electron-builder.

  • Manta : A desktop application for creating invoices with beautiful and customizable templates.

  • Xray : Xray is an experimental Electron-based text editor informed by what we’ve learned in the four years since the launch of Atom.

  • lx-music-desktop : 一个基于 Electron + Vue 开发的音乐软件。

  • marktext : A simple and elegant markdown editor, available for Linux, macOS and Windows.

  • Motrix : A full-featured download manager.

  • BetterDiscord: Better Discord enhances Discord desktop app with new features.

  • windows95 : This is Windows 95, running in an Electron app. Yes, it’s the full thing. I’m sorry.

  • electron-with-express : A simple project demonstrating how to spawn an Express app from Electron as well as providing server logs directly in the Electron app.

  • 2022-Min : Min is a fast, minimal browser that protects your privacy. It includes an interface designed to minimize distractions, and features such as.

  • tools-ocr: 树洞 OCR 文字识别（一款跨平台的 OCR 小工具）

Boilerplate

  • electron-react-typescript-webpack-boilerplate : Pre-configured boilerplate for Electron + React + TypeScript + Webpack.

  • 2020-Svelte NodeGUI : Create native apps for Windows, macOS and Linux using Svelte and CSS.

  • secure-electron-template : A current electron app template with the most popular frameworks, designed and built with security in mind. (If you are curious about what makes an electron app secure, please check out this page).

  • 2022-electron-egg : A simple, cross-platform, enterprise-level desktop software development framework

Development Tools

  • 2018-Electron Fiddle : Electron Fiddle lets you create and play with small Electron experiments. It greets you with a quick-start template after opening – change a few things, choose the version of Electron you want to run it with, and play around.

  • 2018-Electron Forge : Electron Forge unifies the existing (and well maintained) build tools for Electron development into a simple, easy to use package so that anyone can jump right in to Electron development. | 非常便捷的开箱即用的 Electron 开发全流程辅助工具。

  • Electron Packager : Package your Electron app into OS-specific bundles (.app, .exe, etc.) via JavaScript or the command line.

  • Electron Builder : A complete solution to package and build a ready for distribution Electron app with “auto update” support out of the box.

  • nucleus: A configurable and versatile update server for all your Electron apps.

Development Libraries

  • electron-json-storage : electron-json-storage implements an API somehow similar to localStorage to write and read JSON objects to/from the operating system application data directory.

  • electron-util : Useful utilities for Electron apps and modules.

  • electron-pdf: 📄 A command line tool to generate PDF from URL, HTML or Markdown files.

  • 2019-electron-unhandled : Catch unhandled errors and promise rejections in your Electron app.
