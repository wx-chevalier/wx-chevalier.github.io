Gateway-OpenSource-List

API Gateway

  • 2001-HAProxy : HAProxy is a free, very fast and reliable solution offering high availability, load balancing, and proxying for TCP and HTTP-based applications.

  • 2013-Tengine : A distribution of Nginx with some advanced features.

  • 2016-Kong : The open-source API Gateway and Microservices Management Layer, delivering high performance and reliability.

  • 2016-VeryNginx : A very powerful and friendly nginx base on lua-nginx-module( openresty ) which provide WAF, Control Panel, and Dashboards.

  • 2018-HTTPS Portal : A fully automated HTTPS server powered by Nginx, Let’s Encrypt and Docker.

  • 2018-Træfik : Træfik (pronounced like traffic) is a modern HTTP reverse proxy and load balancer made to deploy microservices with ease.

  • 2018-ShenYu : High-Performance Java API Gateway.

  • 2018-Tyk : Tyk Open Source API Gateway written in Go, supporting REST, GraphQL, TCP and gRPC protocols

  • 2020-APISIX : APISIX provides rich traffic management features such as load balancing, dynamic upstream, canary release, circuit breaking, authentication, observability, and more.

Load Balancer & Traffic Orchestration

  • glb-director : GitHub Load Balancer Director and supporting tooling.

  • easegress : A Cloud Native traffic orchestration system

Long Connection Gateway

  • 2015-CometD : The CometD project, a scalable comet (server push) implementation for web messaging.

  • 2016-deepstream : deepstream is an open source server inspired by concepts behind financial trading technology. It allows clients and backend services to sync data, send messages and make rpcs at very high speed and scale.

  • 2017-iot_push : 基于 netty + mqtt 3.1.1 协议开发的物联网消息推送框架。
