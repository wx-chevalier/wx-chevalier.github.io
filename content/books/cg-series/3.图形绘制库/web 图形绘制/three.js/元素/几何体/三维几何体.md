# 三维几何体

# BoxBufferGeometry, BoxGeometry

```js
const width = 8;
const height = 8;
const depth = 8;
const geometry = new THREE.BoxBufferGeometry(width, height, depth);

const width = 8;
const height = 8;
const depth = 8;
const widthSegments = 4;
const heightSegments = 4;
const depthSegments = 4;
const geometry = new THREE.BoxBufferGeometry(
  width,
  height,
  depth,
  widthSegments,
  heightSegments,
  depthSegments
);
```
